# portfolio3

A modern, highly accessible multi-page personal portfolio website built with strict adherence to modern HTML5 semantic standards and WCAG 2.1 accessibility guidelines.

## Features
* **Semantic HTML5 Elements**: Proper tag selection (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`) to construct robust landmark paths for assistive devices.
* **WCAG 2.1 AA/AAA Focus**: Custom visible focus outlines, keyboard menu trapping, skip-links, and high color contrasts (>8.3:1).
* **ARIA Live Regions**: Dynamic announcements for search/filters and contact form submissions.
* **Responsive Architecture**: Mobile-first design using flexbox and grid layouts.
